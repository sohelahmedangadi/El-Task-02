# SA To-Do List App

## Description

A simple Python command-line To-Do List application that allows users to add, remove, and view tasks. Tasks are saved in `ToDo.txt`.

## Features

* Add tasks
* Remove tasks
* View tasks
* Save tasks to file

## How to Run

```
python todo.py
```

## Concepts Used

* Lists
* Functions
* Loops
* File Handling
* enumerate()

Language: Python
